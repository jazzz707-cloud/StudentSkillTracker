# Student Skill Management System

## Overview

The **Student Skill Management System** is a web-based application
developed using **Flask**, **Python**, **HTML**, **CSS**, and
**OpenPyXL**. It allows users to manage student skill records,
automatically generate unique Skill IDs, and store information in an
Excel file.

## Features

-   Add student details
-   Generate unique Skill IDs automatically
-   Store records in an Excel workbook
-   View all student records in a table
-   Simple and user-friendly interface

## Tech Stack

-   Python
-   Flask
-   HTML5
-   CSS3
-   OpenPyXL
-   Excel (.xlsx)

## Installation

1.  Clone the repository:

``` bash
git clone https://github.com/yourusername/Student-Skill-Management-System.git
```

2.  Navigate to the project:

``` bash
cd Student-Skill-Management-System
```

3.  Install dependencies:

``` bash
pip install -r requirements.txt
```

4.  Run the application:

``` bash
python app.py
```
